# Contributing Guide

Thank you for contributing! 🎉

## How to Contribute

1. Fork this repository
2. Create a branch: `git checkout -b feature/your-contribution`
3. Make your changes following the 14-section lesson format
4. Test all code
5. Submit a Pull Request

## 14-Section Lesson Format

Every lesson must include:
1. Problem Statement
2. Business Context
3. Why This Matters
4. Dataset
5. Step-by-Step Solution
6. Alternative Solutions
7. AI-assisted Solution
8. Common Mistakes
9. Interview Questions
10. Practice Exercises
11. Advanced Challenge
12. References
13. Cheat Sheet
14. Summary

## Standards

- Follow the existing folder structure
- Add business context to every example
- Test all code before submitting
